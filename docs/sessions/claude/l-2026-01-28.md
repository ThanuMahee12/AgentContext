## Glob vs Regex Reference

| Shell | Syntax |
|-------|--------|
| Bash glob | `{dif,out}` |
| Regex | `(dif\|out)` |

**Example:**
```bash
# Correct (bash brace expansion)
file /sf/data/bloomberg/back_office_preferred_exch/1.0/raw/2026/20260126/pfd_exch_namr_cins_bbid.{dif,out}

# Wrong (regex syntax in bash)
file /sf/data/bloomberg/back_office_preferred_exch/1.0/raw/2026/20260126/pfd_exch_namr_cins_bbid.(dif|out)
```
