# Brainstorms

Ideas, experiments, and new tool explorations.

## Categories

### New Tools
*Ideas for new tools and utilities*

### Features
*Feature ideas for existing projects*

### Experiments
*Experimental concepts and prototypes*

---

## How to Add

Create a new markdown file:
```
Brainstorms/
├── new-tools/
│   └── my-tool-idea.md
├── features/
│   └── feature-idea.md
└── experiments/
    └── experiment-1.md
```
